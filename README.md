# Project 38: Understanding Human Emotions and Empathy in Disaster Communication

Reproduction instructions: 


Conda environment must have the necessary packages installed.

Create environment on python = 3.10.
Inside the environment run:
  -  python -m spacy download en_core_web_sm
  -  pip install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu121
  -  pip install -r requirements.txt


Task 11.
Inside windows command prompt at the location of the project folder:
  - pip install streamlit
  - pip install plotly
  - streamlit run task11.py
